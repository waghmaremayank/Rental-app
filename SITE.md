# GearShare — Site Vision & Roadmap

## 1. Overview
GearShare is a community rental marketplace mobile-first web app where neighbors can rent and list gear (cameras, e-bikes, tools, projectors, etc.) in Nagpur, India.

## 2. Tech Stack
- Plain HTML + Tailwind CSS (CDN)
- Plus Jakarta Sans (Google Fonts)
- Material Symbols Outlined (Google Icons)
- Python HTTP server on port 3000
